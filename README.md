# RovixGallery - رویکس گالری

Modern **Watch Store E‑commerce** built with **Next.js** with a complete **Admin Dashboard** for managing products and orders.

---

## Tech Stack

- Next.js 16.2
- TypeScript
- Node.js
- MongoDB
- TanStack Query
- Radix UI
- REST API
- API Routes

---

## Features

- Authentication (User / Admin)
- Admin Dashboard
- Product CRUD Management
- Orders Management
- Product Filtering
- Searchbar
- Fully Responsive (Mobile & Desktop)
